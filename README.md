# ADAGE - ADaptive Analysis of Genetic Elements

Presented at the 2019 Canada-Wide Science Fair, and at the 2019 Sanofi Biogenius (Canadian Final).

### Quickstart

```
pip install -r requirements.txt
python app.py
```
